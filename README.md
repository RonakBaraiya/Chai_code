# Chai_code
Chai_code is an full stack project where developer community can connect to same chatroom. 
Tech used: 
  front-end:
    - HTML 5
    - CSS 3
    - Java Script 
      - React

   Back-end:
    - Python
      - Flask, Flask-socketIO ( web socket)

   Database :
    - Postgresql (pg Admin 4)
